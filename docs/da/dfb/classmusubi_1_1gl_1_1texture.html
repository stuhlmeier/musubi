Detailed Description

A wrapper for an OpenGL texture name.

This class additionally contains a flag representing whether the texture should be vertically flipped prior to rendering; this is usually true for textures loaded from disk.

Texture objects do not store the contents of the buffer/pixmap used to create them and thus have no internal state.

Textures are considered "valid" if they contain an OpenGL texture name (see is_valid() const). Valid textures are guaranteed to point to an existing texture, as long as the texture name has not manually been deleted.
