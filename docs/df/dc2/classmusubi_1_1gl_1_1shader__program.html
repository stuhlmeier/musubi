Detailed Description

A wrapper for an OpenGL shader program.

Shader objects do not store the GLSL source code used to create them and thus have no internal state.

Shaders are considered "valid" if they contain an OpenGL shader program handle (see is_valid() const). Valid shaders are guaranteed to point to an existing shader program, as long as the shader program has not manually been deleted.
