Detailed Description

template<pixmap_format Format, typename Traits = pixmap_traits<Format>>
class musubi::buffer_pixmap< Format, Traits >

A memory-backed pixmap.

Template Parameters
Formatthis pixmap's data format
Traitsthe pixmap_traits for this pixmap's data

A backing buffer for a new, empty pixmap is not allocated until an attempt is made to actually modify the underlying buffer.
